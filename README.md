# TransGaspi SpA

Sitio web oficial para **TransGaspi SpA**, empresa líder en el rubro de transportes y servicios, especializada en el arriendo de maquinaria pesada para proyectos de construcción.

## 🚀 Enlace al proyecto en vivo
[https://transgaspi.vercel.app/](https://transgaspi.vercel.app/)

## ✨ Características principales
*   **Hero Cinematográfico**: Fondo de video de alta calidad con degradado y zoom adaptativo para una experiencia visual inmersiva.
*   **Carrusel de Trabajos**: Sección de "Nuestros Trabajos" con movimiento horizontal infinito para exhibición de proyectos.
*   **Diseño Responsivo**: Adaptación total a dispositivos móviles con menús de navegación transparentes y diseño optimizado.
*   **Branding Corporativo**: Logo vectorizado (SVG) integrado para máxima nitidez y rendimiento.
*   **Integración WhatsApp**: Botones de contacto directo configurados para una gestión de cotizaciones eficiente.

## 🛠️ Stack Tecnológico
- **Frontend**: React.js
- **Tooling**: Vite
- **Estilos**: CSS puro (Responsive Design, Flexbox, Grid)
- **Despliegue**: Vercel
